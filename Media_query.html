Media Query

O que é a Responsividade com o Media Query?

A Responsividade é o fato de você conseguir acessar o site criado por meio de diversos tipos de dispositivos diferentes, como no celular, tablet, computador ou notbook, sem que haja uma desconfiguração da página.

Para que essa configuração seja possível e mais simples, pode-se utilizar uma função do CSS chamado "Media Query", que faz com que a página seja acessível em qualquer dispositivo, sendo possível alterar as propriedadas do site quando ele está com um tamanho específico.

Como utilizar o Media Query?

Não é difícil utilizar o Media Query, só é necessário saber estruturá-lo da maneira correta e ele funcionará perfeitamente.

Logo abaixo temos um exemplo de Media Query sendo implementado em um css de uma página. 

@media (max-width: 600px){
    body{
        background-color: blue;
    }
}

Esse é um exemplo simples do uso do Media Query. Nesse exemplo temos:

max-width:600px que serve para determinar os parâmetros em que esse Media Querry funcionará. Ou seja, nesse caso ele irá funcionar quando o tamanho da tela for menor que 600px. E dentro desse @media é possível realizar as mudanças necessárias para que a página continue configurada quando aberta em outros tipos de dispositivos. No caso acima está sendo dito que quando a página estiver com 600px ou menos, o fundo do corpo da página terá a cor azul.

Como se dá para perceber no exemplo acima, a principal informação necessária para que o Media Query funcione, é passando os parâmetros de tamanho da página, e para passá-los podemos utilizar o "max-width:" e/ou o "min-width:" e os seus valores.

Ao utilizar o Media Query, normalmente são utilizados vários operadores lógicos, como o and, not e o only.
